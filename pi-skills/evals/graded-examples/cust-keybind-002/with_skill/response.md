According to `pi-mono/packages/coding-agent/docs/keybindings.md`, keybinding IDs use a `tui.` namespace format. The submit action is `tui.input.submit` (default: `enter`). To remap it to Ctrl+Enter, create or edit `~/.pi/agent/keybindings.json` with: `"tui.input.submit": ["ctrl+enter"]`. Run `/reload` in pi to apply without restarting. The key format uses modifier prefixes like `ctrl+`, `shift+`, `alt+` followed by the key name.
