Per **`pi-mono/packages/web-ui/README.md`** Quick Start: **`SettingsStore`**, **`SessionsStore`**, and **`ProviderKeysStore`** are wired to an **`IndexedDBStorageBackend`** (constructor takes `stores` built from **`settings.getConfig()`**, **`providerKeys.getConfig()`**, **`sessions.getConfig()`**, plus **`SessionsStore.getMetadataConfig()`**). Styles require importing **`@mariozechner/pi-web-ui/app.css`** (shown in the README snippet as `import '@mariozechner/pi-web-ui/app.css';`).
