According to `pi-mono/packages/mom/docs/sandbox.md`, host mode is the default (`mom ./data` runs on host with no isolation). Docker mode (`--sandbox=docker:mom-sandbox`) is the recommended option for security. In Docker mode, all tool execution (bash, read, write, edit) happens inside a container where only `/workspace` (your data dir) is accessible. Host mode runs commands directly on the host machine, meaning anyone in your Slack workspace could potentially execute arbitrary commands and access your files. Use `--sandbox=docker:<name>` unless you have a specific reason to allow host access.
