Skill commands are expanded in `pi-mono/packages/coding-agent/src/core/agent-session.ts` by **`_expandSkillCommand`**. For input like `/skill:my-skill extra args`, it parses the name after `/skill:`, loads the skill’s **`SKILL.md`** from disk with **`readFileSync(skill.filePath, ...)`**, strips front matter, wraps the body in a `<skill>...</skill>` block, and — when there are trailing words after the command — **appends them after that block** (`return args ? \`${skillBlock}\n\n${args}\` : skillBlock`).
