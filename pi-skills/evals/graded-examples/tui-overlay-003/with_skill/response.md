According to `pi-mono/packages/tui/README.md`, you show a modal overlay dialog using `tui.showOverlay(component, options)`. The options object controls positioning, sizing, and whether the overlay is modal (capturing all input). For positioning, you can use an `anchor` such as `center` or `bottom-right`, or provide explicit `row` / `col` values and margins. Overlays are rendered on top of the main component tree and are managed by the TUI's differential rendering system, so they integrate seamlessly without manual screen clearing.
