According to `pi-mono/packages/tui/README.md`, you create a custom component by implementing the `Component` interface. The key method is `render(width)`, which must return lines that never exceed the given `width` parameter — use `truncateToWidth` or `wrapTextWithAnsi` to enforce this. For keyboard input, use `matchesKey(data, Key.xxx)` in your `handleInput` method rather than manual string comparison, as it handles modifier keys and terminal escape sequences correctly.
