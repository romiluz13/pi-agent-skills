According to `pi-mono/packages/mom/docs/events.md`, you schedule a periodic task by writing a JSON event file to `workspace/events/`. The event JSON uses these fields: `type: "periodic"`, `channelId` (the Slack channel ID, e.g., `"C123ABC"`), `text` (the instruction for Mom), `schedule` (standard cron syntax, e.g., `"0 9 * * 1-5"` for weekdays at 9 AM), and `timezone` (IANA timezone name, e.g., `"Europe/Vienna"`). Each event file must have a unique descriptive filename ending in `.json`. A maximum of 5 events can be queued per channel.
