According to `pi-mono/packages/pods/README.md`, you start a Qwen model on a multi-GPU pod using the full Hugging Face path: `pi start Qwen/Qwen2.5-Coder-32B-Instruct --name qwen`. Pi automatically assigns the model to available GPUs when running multiple models on the same pod. For tool-call parsing, pi configures the `hermes` parser for most Qwen models. The exception is Qwen3-Coder, which uses the `qwen3_coder` parser instead. You do not need to manually set up tool-calling parsers for predefined models.
