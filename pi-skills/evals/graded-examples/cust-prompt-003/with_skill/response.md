According to `pi-mono/packages/coding-agent/docs/prompt-templates.md`, prompt templates are loaded from global (`~/.pi/agent/prompts/*.md`), project (`.pi/prompts/*.md`), packages, settings paths, and CLI `--prompt-template`. Templates are Markdown files with optional YAML frontmatter (e.g., `description`). The filename minus `.md` becomes the `/name` command. Arguments use positional syntax: `$1`, `$2` for specific args, `$@` for all args joined, and `${@:N}` for args from the Nth position. There is no documented include directive in the current prompt-template docs.
